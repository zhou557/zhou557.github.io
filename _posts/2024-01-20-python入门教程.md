## Python 简介

Python 是一种高级编程语言，以其简洁的语法和强大的功能而闻名。它广泛应用于 Web 开发、数据分析、人工智能等领域。

## 环境搭建

首先，您需要安装 Python：

1. 访问 [Python 官网](https://www.python.org/)
2. 下载适合您系统的版本
3. 安装并配置环境变量

## 第一个 Python 程序

```python
print("Hello, World!")
```

运行这段代码，您将看到输出：`Hello, World!`

## 基本数据类型

Python 有以下基本数据类型：

- **整数**：`int`
- **浮点数**：`float`
- **字符串**：`str`
- **布尔值**：`bool`

## 条件语句

```python
if x > 0:
    print("x 是正数")
elif x == 0:
    print("x 等于 0")
else:
    print("x 是负数")
```

## 循环

```python
# for 循环
for i in range(10):
    print(i)

# while 循环
while x < 10:
    print(x)
    x += 1
```
